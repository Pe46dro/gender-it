# gender-it

Guess the gender of an italian name

### Usage
```javascript
var gender_it = require('gender-it')

gender_it.guess('Giacomo')
// => 'm'
gender_it.guess('Chiara')
// => 'f'
gender_it.guess('Voldemort')
// => undefined
```
