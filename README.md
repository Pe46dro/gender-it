# gender-it

Guess the gender of an italian name

### Usage
```javascript
var gender_it = require('gender-it')
gender.guess('Giacomo')
// => 'm'
gender.guess('Chiara')
// => 'f'
gender.guess('Voldemort')
// => undefined
```
